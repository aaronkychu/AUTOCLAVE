# Autoclave Logging System

This project provides a web-based solution for tracking autoclave usage and sterilization cycles in a clinical setting. It includes database setup scripts, PHP integration, and a structured directory layout for deployment.

---

## 📦 Database Setup

1. Import the provided **SQL.sql** file into **phpMyAdmin** to create the database and tables.
2. Using **phpMyAdmin**, configure the `autoclaves` table to register autoclaves for your clinic.
3. Ensure that the database is properly initialized before running the application.

---

## 📂 File Organization

- All project files should be placed in the **root directory** of your web server.
- Create a dedicated folder (e.g., `/autoclave`) to store and serve the application.

---

## ⚙️ Deployment Notes

- Compatible with **PHP** and **MySQL** environments.
- Designed to run seamlessly on desktop browsers, iPads, and iPhones.
- Ensure proper permissions are set for the web server directory.

---

## 🚀 Getting Started

1. Clone or download this repository.
2. Place the files in your web server’s root directory under `/autoclave`.
3. Import `SQL.sql` into **phpMyAdmin**.
4. Access the application via your browser to begin logging autoclave cycles.
